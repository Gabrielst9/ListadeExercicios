# Academia do Programador
## Lista de Exercícios

Lista de Exercício do 1 até o 28 da Academia do Programador. Todos os exercícios foram feitos em C#.
Presente conteúdos de Laços de Repetição, Condicionais...

## Como executar

### Clone o repositório

```bash
git clone
```

# Entre na pasta do projeto

```bash
cd AcademiaDoProgramador
```

# Execute o projeto

```bash
dotnet run
```

## Requisitos

- Visual Studio 2022
- .NET 8.0
